# Repetitive Flight Plan (RPL)


## Definition
An RPL is a flight plan related to a series of frequently recurring, regularly operated individual flights with identical basic features, submitted by an operator for retention and repetitive use by ATS.

## Criteria for Use
*   Typically for flights operated on the same day(s) of 10 or more consecutive weeks.
*   Must be submitted at least **2 weeks** before the first flight of the series is effective.
*   Used mainly for scheduled airline operations to reduce paperwork and transmission load.

## Changes to RPL
*   **Permanent Changes**: Must be submitted to the RPL collection agency.
*   **Temporary Changes (Single Flight)**:
    *   If a specific flight in the series needs to be changed for one day (e.g., diff aircraft, diff route), a standard FPL might be filed to overwrite it, or a Change (CHG) message sent.
    *   If the flight is cancelled, a Cancel (CNL) message is required.
