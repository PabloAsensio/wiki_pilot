title: "Aviso de Configuración de Despegue (TOCW)"
---

## Descripción General

El sistema de **Aviso de Configuración de Despegue (Take-off Configuration Warning - TOCW)** es un mecanismo de seguridad crítico diseñado para evitar que una aeronave intente despegar en una configuración insegura.

## Regulaciones (CS 25.703)

Las regulaciones exigen que las grandes aeronaves de transporte estén equipadas con un sistema que proporcione una advertencia **auditiva** automáticamente durante la **porción inicial de la carrera de despegue**.

*   **Momento:** La alerta debe activarse cuando **se avanzan los aceleradores** (aplicación de potencia de despegue) mientras la aeronave está en tierra.
*   **Indicación:** El requisito mínimo es una alerta **Auditiva** (p. ej., Sirena, Bocina o Voz "CONFIG FLAPS"). Muchas aeronaves también proporcionan alertas visuales (Master Warning Rojo + Mensaje).

## Condiciones de Activación

El sistema se activa si cualquiera de los siguientes parámetros no está en la posición aprobada para el despegue:

1.  **Flaps / Slats:** No están dentro del rango aprobado para el despegue (o no están ajustados).
2.  **Compensador Longitudinal (Estabilizador):** No está en la banda verde (Rango de Despegue).
3.  **Spoilers / Frenos Aerodinámicos:** Desplegados o no en la posición segura de despegue.
4.  **Freno de Estacionamiento:** Puesto o no liberado.
5.  **Controles de Vuelo:** Bloqueos de control enganchados (si aplica).
6.  **Puertas:** Puertas externas esenciales no cerradas y bloqueadas (en algunos tipos).

## Uso Operativo

*   **Función de Prueba:** Las aeronaves modernas (como el Airbus A320) incluyen un botón pulsador `T.O CONFIG`. Al presionarlo se simula la aplicación de potencia de despegue para verificar si la configuración actual es válida. Si lo es, se muestra un mensaje específico (p. ej., **"TO CONFIG NORMAL"**).
*   **Inhibición:** La advertencia se inhibe típicamente después de una cierta velocidad ($V_1$ o $V_R$) para priorizar efectivamente las alertas y evitar distracciones durante la fase crítica de rotación.
