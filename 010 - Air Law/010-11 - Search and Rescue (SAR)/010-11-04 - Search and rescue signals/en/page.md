# Search and Rescue Signals: Ground-Air Codes and Acknowledgement Procedures

## Ground-Air Visual Signals
These signals are used by survivors and rescue units to communicate with aircraft. They must be large and have contrast.

![Ground-Air Visual Code for Use by Survivors](https://upload.wikimedia.org/wikipedia/commons/e/e0/Ground-Air_Visual_Code_for_Use_by_Survivors.png)
*Standard code used by survivors. Note 'V' (Require assistance) and 'X' (Require medical assistance).*

**Code for Survivors:**
*   **V**: Require assistance.
*   **X**: Require medical assistance.
*   **N**: No / Negative.
*   **Y**: Yes / Affirmative.
*   **Arrow (➔)**: Proceeding in this direction.

**Code for Rescue Units:**
*   **LLL**: Operation completed.
*   **LL**: All personnel found.
*   **++**: Only some personnel found.

### Air-Ground Signals
To indicate that the ground signal has been **understood**:
*   **By day:** Rocking wings.
*   **By night:** Flashing twice the landing lights or navigation lights.

The **absence** of these signals indicates that the ground message has **not** been understood.
