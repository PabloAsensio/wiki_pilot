## Resumen de Distancias Declaradas

| Distancia | Definición | Fórmula (Simplificada) |
| :--- | :--- | :--- |
| **TORA** | Take-Off Run Available | Pista física apta para rodaje de despegue |
| **TODA** | Take-Off Distance Available | TORA + Clearway (CWY) |
| **ASDA** | Accelerate-Stop Distance Available | TORA + Stopway (STW) |
| **LDA** | Landing Distance Available | Pista disponible para aterrizaje (afectada por umbral desplazado) |

*   **TODA incluye el Clearway** (puede ser sobre agua).
*   **ASDA incluye el Stopway** (pavimento resistente para frenado de emergencia).
*   Si no hay Stopway, ni Clearway, ni umbral desplazado: **TORA = TODA = ASDA = LDA**.

## Luces de Aproximación
La medición de las luces de aproximación siempre se realiza desde el **umbral (threshold)** hacia afuera.

### Colores de Señales (Resumen)
*   **Señales de Instrucción Obligatoria:** Blanco sobre Rojo (Ej. STOP, Entrada Prohibida).
*   **Señales de Ubicación:** Amarillo sobre Negro.
*   **Señales de Dirección/Destino:** Negro sobre Amarillo.
